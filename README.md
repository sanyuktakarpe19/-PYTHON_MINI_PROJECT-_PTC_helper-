# Plant Tissue Culture Assistant

A beginner-level, menu-driven Python mini project.

## Features
- Lists plant tissue culture media
- Suggests suitable media
- Calculates media preparation for any volume

## Media Included
Knudson, LS, NN, White, DKW, QL, SH, Knop, MS, WPM, B5, N6

## How to Run
- Open the Python file
- Run the program
- Use the menu options

## Requirements
- Python 3.x
